# Trivo 3.0 - Unity 스타일 도구 UI 추가

## 🎯 주요 신기능

### 1️⃣ 도구 패널 (Tools Panel)
**위치**: 사이드바 → "🔧 도구 / Tools" 탭

도구 목록이 한눈에 보이는 패널입니다:

```
┌─────────────────────┐
│ 🔧 도구 / Tools     │
├─────────────────────┤
│ 👆 선택 (Q)  [●]   │  <- 현재 선택됨
│ ➡️  이동 (G)         │
│ 🔄 회전 (R)         │
│ 📏 크기 (S)         │
├─────────────────────┤
│ ℹ️ 정보              │
│ 모드: 선택           │
│ (도구별 설명)        │
├─────────────────────┤
│ ⌨️ 단축키            │
│ Q - 선택            │
│ G - 이동            │
│ ... 등등            │
└─────────────────────┘
```

**특징:**
- 👆 **선택 (Q)**: 마우스로 모델 선택
- ➡️  **이동 (G)**: 위치 조정
- 🔄 **회전 (R)**: 방향 설정
- 📏 **크기 (S)**: 스케일 조정

**사용법:**
```
1. 도구 버튼 클릭 (또는 단축키)
2. 마우스로 씬 조작
3. 다른 도구 선택 또는 Esc로 취소
```

---

### 2️⃣ 텍스처 패널 (Texture Panel)
**위치**: 사이드바 → "🎨 텍스처 / Texture" 탭

텍스처 페인팅 도구입니다:

```
┌──────────────────────┐
│ 🎨 페인팅 / Painting │
├──────────────────────┤
│ 색상: [████]         │
│ [색상 선택 / Pick]   │
│                      │
│ 크기: [━━━━━━] 10px  │
│                      │
│ [🖌 페인팅 시작]    │
└──────────────────────┘
│                      │
│ 🖼 텍스처 / Preview  │
├──────────────────────┤
│     [색상 프리뷰]    │
│                      │
│ 모델: character.fbx  │
│ 정점 색상: 25000개   │
│ 텍스처: 미지원 (향후)│
└──────────────────────┘
```

**기능:**
- 🎨 **색상 선택**: 브러시 색상 변경
- 📏 **크기 조정**: 브러시 크기 (1-50px)
- 🖌 **페인팅 시작**: 모드 활성화
- 👁️ **프리뷰**: 현재 설정 미리보기

**사용법:**
```
1. 모델 선택 (Tools 패널에서)
2. 색상 및 크기 설정
3. "페인팅 시작" 버튼 클릭
4. 씬에서 모델 표면 드래그로 칠하기
5. "페인팅 시작" 다시 클릭해서 종료
```

**향후 기능:**
- ✅ 정점 색상 페인팅
- 🔜 텍스처 파일 직접 편집
- 🔜 여러 레이어 지원
- 🔜 실행 취소 / 재실행

### 3️⃣ 스카이박스 패널 (Skybox Panel)
**위치**: 사이드바 → "🌅 스카이박스 / Skybox" 탭

배경 환경 설정 도구입니다:

```
┌─────────────────────────┐
│ 🌅 스카이박스 / Skybox  │
├─────────────────────────┤
│ 타입: [💜 단색 ▼]      │
├─────────────────────────┤
│ 단색 설정:              │
│ 색상: [████] [선택]    │
├─────────────────────────┤
│ 공통 설정:              │
│ 밝기: [━━━━━] 1.0     │
│ 회전: [━━━━━] 0°      │
└─────────────────────────┘
```

**스카이박스 타입:**

#### 1. 💜 단색 (Solid Color)
```
간단하고 빠른 배경
- 예: 파란색 하늘, 흰색 배경
- 용도: 카탈로그, 제품 뷰
- 성능: 매우 빠름
```

#### 2. 🌈 그라디언트 (Gradient)
```
하늘-지면 효과
- 상단색: 하늘색 (보통)
- 하단색: 지면색 (모래색 등)
- 용도: 자연스러운 배경
- 성능: 빠름
```

#### 3. 🖼 텍스처 (Texture - 향후)
```
환경 맵핑 (IBL)
- 정육면체 맵 (Cubemap)
- HDR 이미지 지원
- 용도: 현실적인 조명
- 성능: 중간
```

**공통 설정:**
- 🎚️ **밝기** (0.0-3.0): 스카이박스 밝기 조절
- 🔄 **회전** (0-360°): Y축 회전

**사용 예:**

```
예 1: 스튜디오 라이팅
- 단색 흰색 (Solid)
- 밝기: 1.2

예 2: 자연 배경
- 그라디언트 (Gradient)
- 상단: 하늘색 (135, 206, 235)
- 하단: 모래색 (255, 228, 181)
- 밝기: 1.0

예 3: 밤하늘
- 단색 짙은 파란색
- 밝기: 0.6
- 회전: 회전해서 별 위치 조절
```

## 🎮 키보드 단축키

### 도구 선택
```
Q - 선택 도구 (Select Tool)
G - 이동 도구 (Move Tool)
R - 회전 도구 (Rotate Tool)
S - 크기 도구 (Scale Tool)
```

### 변환 모드
```
마우스 드래그- 현재 도구로 조작
좌클릭 - 조작 확정
Esc - 조작 취소
```

### 카메라
```
F - 씬 맞춤 (Fit Scene)
마우스 좌클릭 드래그 - 궤도 회전
마우스 우클릭+Shift - 패닝
마우스 휠 - 줌
```

## 💡 사용 팁

### Tip 1: 효율적인 작업 흐름
```
1. "🔧 도구" 탭에서 현재 도구 확인
2. 도구 버튼 클릭 또는 단축키 입력
3. 씬에서 모델 조작
4. 다른 도구로 전환 반복
5. "🌅 스카이박스"에서 배경 조정
```

### Tip 2: 색상 선택
```
텍스처 패널의 [색상 선택] 버튼
→ 색상 선택 대화 표시
→ 편안한 색상 선택
→ 프리뷰에서 즉시 확인
```

### Tip 3: 스카이박스 조합
```
좋은 조합 1: 단색 흰색 + 강한 라이트
→ 전문적인 제품 쇼케이스

좋은 조합 2: 그라디언트 + 자연광 프리셋
→ 실외 장면 시뮬레이션

좋은 조합 3: 어두운 색 + 낮은 밝기
→ 드라마틱한 조명
```

### Tip 4: 패널 전환
```
탭 클릭으로 빠르게 전환
- 도구 조정: "🔧 도구"
- 모델 관리: "씬"
- 색상 칠하기: "🎨 텍스처"
- 배경 설정: "🌅 스카이박스"
- 조명/환경: "환경"
```

## 🔧 기술 세부사항

### 파일 구조
```
src/ui/
  ├── ToolsPanel.h/.cpp      # 도구 목록 패널
  ├── TexturePanel.h/.cpp    # 텍스처 페인팅 패널
  ├── SkyboxPanel.h/.cpp     # 스카이박스 설정 패널
  └── Sidebar.h/.cpp         # 메인 사이드바 (모든 패널 포함)
```

### 신호/슬롯
```cpp
// ToolsPanel
signal: toolSelected(const QString &toolName)

// TexturePanel
signal: paintingModeChanged(bool enabled)
signal: brushColorChanged(const QColor &color)
signal: brushSizeChanged(int size)

// SkyboxPanel
signal: skyboxTypeChanged(int type)
signal: skyboxColorChanged(const QColor &color)
signal: skyboxIntensityChanged(double intensity)
signal: skyboxRotationChanged(double rotation)
```

### Viewport와의 통합
```cpp
// Viewport는 다음을 처리합니다:
// 1. 도구 신호 수신 → 변환 모드 설정
// 2. 페인팅 신호 수신 → 텍스처 업데이트
// 3. 스카이박스 신호 수신 → 배경 렌더링 업데이트
```
